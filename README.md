# Noam Pony Portfolio

Personal portfolio website for Noam Pony, built with Next.js App Router,
TypeScript, Tailwind CSS, ESLint, and pnpm.

## Requirements

- Node.js 20 or newer
- pnpm 9 or newer

## Commands

Install dependencies:

```bash
pnpm install
```

Start the local development server:

```bash
pnpm dev
```

Create a production build:

```bash
pnpm build
```

Run linting:

```bash
pnpm lint
```

Run the TypeScript type-check:

```bash
pnpm type-check
```
